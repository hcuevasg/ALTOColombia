Cómo se calcula (metodología ALTO)

Medimos cada hecho negativo sobre el cliente en prensa abierta. Cada hecho suma según:

  • Gravedad (leve / moderada / grave): qué tan dañino es para la reputación.
  • Cobertura: cuántos medios distintos lo replican — más cobertura, más impacto, con un tope para que la repetición de una misma noticia no infle el índice.

Las menciones positivas o neutras no suman. El nivel (Bajo / Moderado / Alto) ubica el total de hoy; lo que motiva una alerta es que el índice suba frente al promedio del cliente, no su valor absoluto.
